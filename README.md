# To-Do Web App

A modern and responsive To-Do Web Application built using HTML, CSS, Bootstrap, and Vanilla JavaScript.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Separate Pending and Completed task sections
- Dynamic task counters
- Responsive design for mobile, tablet, and desktop
- Modern dark UI design

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (DOM Manipulation)

## Project Structure

```bash
todo-webapp/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run the Project

1. Download or clone the repository

```bash
git clone https://github.com/yourusername/todo-webapp.git
```

2. Open the project folder

3. Run `index.html` in your browser
